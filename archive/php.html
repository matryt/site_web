PHP est un langage de programmation libre, principalement utilisé pour produire des pages Web dynamiques via un serveur HTTP, mais pouvant également fonctionner comme n'importe quel langage interprété de façon locale. PHP est un langage impératif orienté objet.

Présentation

PHP est un langage de script très souvent utilisé pour les serveurs : il interprète alors le code PHP des pages web demandées par les utilisateurs et renvoie code et données interprétables par les navigateurs web. C'est un langage peu typé et souple. Il fonctionne sur toutes les systèmes d'exploitation connus.

Histoire

Le langage est créé en 1994 par Rasmus Lerdorf pour son site web. C'était au début une bibliothèque en C qu'il utilisait pour garder une trace des visiteurs ayant consulté son CV. Au fur et à mesure des ajouts, Rasmus a transformé la bibliothèque en inmplémentation capable d'interagir avec des bases de données et créer des applications pour le Web. En 1997, Andi Gutmans et Zeev Suraski redéveloppent le coeur du langage et commencent la réécriture interne de son moteur : leur version a servi de base à la v4 de PHP.

Fonctionnement

PHP est descendant du C, et leurs syntaxes sont similaires. Lorsqu'un visiteur veut consulter une page web, son navigateur envoie une requête au serveur HTTP. Si la page est un script PHP, celui-ci appelle l'interpréteur PHP qui traite et génère le code de la page, qu'il renvoie au serveur, qui la met à disposition du navigateur.
